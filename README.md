# 🧠 QuizVerse

QuizVerse is a modern and responsive quiz web application built with **HTML5**, **CSS3**, and **JavaScript**. The project provides an engaging platform for users to test their knowledge through multiple quiz categories with a clean interface and smooth user experience.

## ✨ Features

- 📚 Multiple Quiz Categories
- 🎯 Real-Time Score Tracking
- 📱 Fully Responsive Design
- ⚡ Fast & Interactive UI
- 🎨 Clean Modern Interface
- 🔄 Dynamic Quiz Navigation
- 🏆 Result & Performance Display

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
